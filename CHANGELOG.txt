Class hierarchy.

DomainInterface.php
  Defines attributes of the Domain Entity and directly
  sets / gets properties of that entity.

  public function isActive();
  public function isDefault();
  public function saveDefault();
  public function enable();
  public function disable();
  public function getProperty();
  public function saveProperty($key, $value);
  public function setPath();
  public function setUrl();
  public function getPath();
  public function getUrl();
  public function getResponse();
  public function setResponse();

DomainResolverInterface.php
  Takes an http Request and returns the proper domain
  entity that governs the request. Called by classes that
  wish to interact with the active domain.

  public function setRequestDomain($httpHost);
  public function resolveActiveDomain(DomainInterface $domain);
  public function resolveActiveHostname();
  public function setHttpHost($httpHost);
  public function getHttpHost();
  public function getActiveDomain();
  public function getActiveId();


DomainLoaderInterface.php
  Loads specific domains based on parameters. E.g. gets the
  default domain object.

  change get to load

  public function loadDefaultDomain();
  public function loadDefaultId();
  public function loadMultiple($ids = array(), $reset = FALSE);
  public function loadMultipleSorted($ids = array());
  public function loadByHostname($hostname);
  public function loadOptionsList();
  private function sort($a, $b);
  public function loadSchema();


DomainCreatorInterface.php
  Helper class for the creation of new domains.

  change get to create or list

  public function createDomain($inherit = FALSE, array $values = array());
  public function createNextId();
  public function createHostname();
  public function createMachineName($hostname);

DomainValidatorInterface.php

  public function validate();
  public function checkResponse();
  public function getRequiredFields();

Negotiation Operations

  Remove the EventListener and make it such that the Active
